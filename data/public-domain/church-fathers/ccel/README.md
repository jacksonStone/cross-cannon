# CCEL Early Church Fathers Downloads

Downloaded from the Christian Classics Ethereal Library Early Church Fathers
index:

```text
https://www.ccel.org/fathers
```

## Contents

This folder contains 38 CCEL volumes:

- `anf01` through `anf10`
- `npnf101` through `npnf114`
- `npnf201` through `npnf214`

Each volume is saved in two raw formats:

```text
raw/txt/<volume-id>.txt
raw/xml/<volume-id>.xml
```

The `source-pages/fathers.html` file is the downloaded CCEL index page used to
discover the volume IDs.

## Formats

- `raw/txt`: plain text generated by CCEL. This is easiest for reading and
  quick review.
- `raw/xml`: CCEL ThML/XML. This preserves more structure and metadata and is
  the better candidate for later parsing into indexed works, sections, and
  citations.

## Source URL Patterns

Plain text:

```text
https://www.ccel.org/ccel/schaff/<volume-id>/cache/<volume-id>.txt
```

ThML/XML:

```text
https://www.ccel.org/ccel/schaff/<volume-id>.xml
```

## Public Domain Note

The downloaded text files identify these volumes as public domain. The XML
metadata records the original 19th-century publication history for the
Ante-Nicene Fathers and Nicene/Post-Nicene Fathers volumes.
